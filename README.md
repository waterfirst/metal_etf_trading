# 🥇 귀금속 ETF 트레이딩 신호 대시보드

금(Gold), 은(Silver), 구리(Copper) ETF의 최적 매수/매도 타이밍을 실시간으로 분석하는 전문 트레이딩 대시보드입니다.

![Python](https://img.shields.io/badge/python-3.8+-blue.svg)
![Streamlit](https://img.shields.io/badge/streamlit-1.32.0+-red.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)

## 📊 주요 기능

### 1. 5단계 통합 트레이딩 신호
- 🟢🟢 **강력 매수** (5점)
- 🟢 **매수** (4점)
- 🟡 **중립** (3점)
- 🔴 **매도** (2점)
- 🔴🔴 **강력 매도** (1점)

### 2. 다층 분석 시스템

#### 📐 핵심 비율 지표
- **금/은 비율 (Gold/Silver Ratio)**
  - 역사적 정상 범위: 68-82
  - 극단 신호: 90 초과(은 저평가), 60 미만(금 저평가)
  - 실전 페어 트레이딩 전략 제공

- **구리/금 비율 (Copper/Gold Ratio)**
  - 경기 순환 온도계
  - 리스크 온/오프 신호
  - 구조적 수요 vs 통화적 수요 분석

#### 💹 개별 금속 모멘텀 분석
- 일간/월간/분기/YTD 성과 추적
- 이동평균선 (MA20, MA50) 분석
- 거래량 추세 모니터링

#### 🌍 거시경제 환경 분석
- 달러지수 (DXY) - 금속 가격 역상관
- 미국 10년물 국채 - 실질금리 영향
- S&P 500 - 리스크 심리
- VIX - 변동성 지표

### 3. 실시간 모니터링
- Yahoo Finance API 실시간 연동
- ETF 현물 + 선물 가격 동시 추적
- 5분 주기 자동 캐싱

### 4. 시각화 차트
- 📈 개별 금속 가격 추이 (이동평균선 포함)
- 📊 거래량 분석
- 🔄 상대 성과 비교 (정규화 차트)
- 📉 백테스트 시뮬레이션

### 5. 실전 트레이딩 가이드
- ⏰ LP 호가 제시 시간 경고 (한국 투자자)
- 💰 분할 매수/매도 전략
- 🛡️ 손절/익절 가이드라인
- ⚠️ ETF 추적오차 주의사항
